Sample sequencer inspired by the Roland TR-808 drum machine.

8 customizable sample pads. Users can upload samples, record with their iPhone microphone, or use the drum sounds from iconic drum machines (Roland TR-808, 505, 909, LinnDrum, UniVox SR-95, etc). Sample pads allow for user set start and end points. Users can pitch or reverse these samples. Once samples are loaded into the pads, the user can sequence these samples in a 16-step step sequencer.

The sequenced audio is run through a 7-band graphic equalizer and then into an effects mixer.
